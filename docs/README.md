# Qulome - 微信文章排版工具

## 版本信息
- 版本：v1.0.0
- 发布日期：2025-07-03
- 大小：356KB

## 功能特性
- 📝 WYSIWYG编辑器，支持富文本编辑
- 🎨 主题系统，支持自定义样式和实时预览
- 🖼️ 图标库管理，支持SVG图标
- 💾 草稿自动保存和发布管理
- 📋 一键复制到微信编辑器

## 技术栈
- HTML5 + CSS3 + JavaScript
- 本地存储（localStorage）
- 模块化架构

## 部署说明
1. 将所有文件上传到Web服务器
2. 确保服务器支持静态文件服务
3. 访问index.html即可使用

## 浏览器兼容性
- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 注意事项
- 需要现代浏览器支持
- 数据存储在本地，清除浏览器数据会丢失
- 建议使用HTTPS协议部署
